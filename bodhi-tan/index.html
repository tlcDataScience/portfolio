Project 1: Time Breakdown (time_breakdown.py)

Prose Description

This project uses arithmetic operations to break down a total number of minutes into days, hours, and minutes. It demonstrates the use of floor division and the modulo operator to handle integer division and remainders.

Logic Flow Diagram

flowchart TD A[Start: Input total minutes] B[Calculate days = total_minutes divided by (24*60) using floor division] C[Calculate remaining hours = (total_minutes mod (24*60)) divided by 60 using floor division] D[Calculate remaining minutes = total_minutes mod 60] E[Display the result] A --> B B --> C C --> D D --> E

Python Code

total_minutes = int(input("Enter the total number of minutes: "))
days = total_minutes // (24 * 60)
remaining_hours = (total_minutes % (24 * 60)) // 60
remaining_minutes = total_minutes % 60
print(f"{total_minutes} minutes is equal to {days} days, {remaining_hours} hours, and {remaining_minutes} minutes.")

Project 4: Coffee Shop Inventory

Prose Description

This project simulates an arithmetic progression for a coffee shop's daily bean inventory. A for loop is used to update and accumulate the stock over 7 days.

Logic Flow Diagram

flowchart TD A[Start: Initialize day 1 beans & daily increase] B[Loop from day 1 to 7] C[Update beans for each day] D[Accumulate total beans] E[Display day 7 beans and total beans] A --> B B --> C C --> D D --> E

Python Code

initial_coffee_beans = 50  # Initial beans on day 1
daily_increase = 4          # Daily increase

coffee_beans_on_day_7 = initial_coffee_beans
total_coffee_beans_after_7_days = 0

for day in range(1, 8):
    coffee_beans_on_day_7 += daily_increase
    total_coffee_beans_after_7_days += initial_coffee_beans + (day - 1) * daily_increase

print(f"Coffee beans on day 7: {coffee_beans_on_day_7}")
print(f"Total coffee beans after 7 days: {total_coffee_beans_after_7_days}")

Project 6: Factorial Calculator (factorial_calculator.py)

Prose Description

This project computes the factorial of a number using iterative multiplication. A for loop multiplies a running product, and input validation ensures only non-negative integers are processed.

Logic Flow Diagram

flowchart TD A[Start: Input a number] B[Check: Is number non-negative?] C[Initialize factorial_result = 1] D[Loop from 1 to number] E[Multiply factorial_result by current number] F[Display factorial_result] A --> B B -- Valid --> C C --> D D --> E E --> F

Python Code

user_input = int(input("Enter a number: "))

if user_input < 0:
    print("Please enter a non-negative integer.")
else:
    factorial_result = 1
    for i in range(1, user_input + 1):
        factorial_result *= i
    print(f"The factorial of {user_input} is {factorial_result}.")
Project 7: Right Triangle Checker (right_triangle_checker.py)

Prose Description

This program verifies if a triangle is right-angled by sorting its sides and using the Pythagorean theorem. It demonstrates list manipulation and arithmetic comparison in Python.

Logic Flow Diagram

flowchart TD A[Start: Input three side lengths] B[Sort sides in ascending order] C[Check: Does side1² + side2² equal side3²?] D[Display if right triangle] A --> B B --> C C --> D

Python Code

side1 = float(input("Enter the length of side 1: "))
side2 = float(input("Enter the length of side 2: "))
side3 = float(input("Enter the length of side 3: "))
sides = [side1, side2, side3]
sides.sort()

if sides[0]**2 + sides[1]**2 == sides[2]**2:
    print("The triangle is a Right Triangle!")
else:
    print("The triangle is not a Right Triangle.")
Project 8: Number Guessing Game (my_number_guessing_game.py)

Prose Description

In this interactive game, a random secret number is generated. A while loop with conditional checks provides hints—indicating whether the guess is too high or too low—until the correct number is guessed.

Logic Flow Diagram

flowchart TD A[Start: Generate secret number (1-100)] B[Loop: Prompt user for guess] C[If guess > secret then display: Too high] D[If guess < secret then display: Too low] E[If guess equals secret then exit loop] F[Display congratulatory message] A --> B B --> C B --> D B --> E C --> B D --> B E --> F

Python Code

import random

secret_number = random.randint(1, 100)
guess = 0

while guess != secret_number:
    guess = int(input("Guess the number (between 1 and 100): "))
    if guess > secret_number:
        print("Too high! Try again.")
    elif guess < secret_number:
        print("Too low! Try again.")

print("Congratulations! You guessed the correct number.")
Project 9: Palindrome Checker (my_palindrome_checker.py)

Prose Description

This project checks whether a word is a palindrome. It reverses the input word using a loop and then compares it to the original, showcasing string manipulation and control flow.

Logic Flow Diagram

flowchart TD A[Start: Input a word] B[Initialize empty reversed_word] C[Loop: Build reversed_word character by character] D[Compare original word with reversed_word] E[Display if palindrome] A --> B B --> C C --> D D --> E

Python Code

word = input("Enter a word: ")
reversed_word = ""

while len(reversed_word) < len(word):
    reversed_word = word[len(reversed_word)] + reversed_word

if word == reversed_word:
    print(f"{word} is a Palindrome!")
else:
    print(f"{word} is not a Palindrome.")
