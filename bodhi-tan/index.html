Project 1: Time Breakdown (time_breakdown.py)

Prose Description

This project uses arithmetic operations to convert a total number of minutes into days, hours, and minutes. It leverages floor division and the modulo operator to break down the input into the proper time units.

Logic Flow Diagram

flowchart TD A[Enter total minutes] B[Calculate days = minutes // (24*60)] C[Calculate remaining hours = (minutes % (24*60)) // 60] D[Calculate remaining minutes = minutes % 60] E[Output result] A --> B B --> C C --> D D --> E

Python Code

total_minutes = int(input("Enter the total number of minutes: "))
days = total_minutes // (24 * 60)
remaining_hours = (total_minutes % (24 * 60)) // 60
remaining_minutes = total_minutes % 60
print(f"{total_minutes} minutes is equal to {days} days, {remaining_hours} hours, and {remaining_minutes} minutes.")

Project 4: Coffee Shop Inventory

Prose Description

This project simulates a coffee shop's bean inventory using an arithmetic progression. A for loop iterates over 7 days to update the stock and accumulate the total, demonstrating loop control and incremental arithmetic.

Logic Flow Diagram

flowchart TD A[Initialize beans = 50, daily increase = 4] B[For day from 1 to 7] C[Calculate day's beans: beans + (day-1)*increase] D[Accumulate total beans] E[Output beans on day 7 and total beans] A --> B B --> C C --> D D --> E

Python Code

initial_coffee_beans = 50  # Initial beans on day 1
daily_increase = 4          # Daily increase

coffee_beans_on_day_7 = initial_coffee_beans
total_coffee_beans_after_7_days = 0

for day in range(1, 8):
    coffee_beans_on_day_7 += daily_increase
    total_coffee_beans_after_7_days += initial_coffee_beans + (day - 1) * daily_increase

print(f"Coffee beans on day 7: {coffee_beans_on_day_7}")
print(f"Total coffee beans after 7 days: {total_coffee_beans_after_7_days}")

Project 6: Factorial Calculator (factorial_calculator.py)

Prose Description

This project computes the factorial of a non-negative integer using iterative multiplication. It checks for valid input, then uses a for loop to multiply a running product.

Logic Flow Diagram

flowchart TD A[Enter a number] B{Is number < 0?} C[Output error: non-negative required] D[Initialize factorial = 1] E[For i from 1 to number] F[Multiply factorial by i] G[Output factorial result] A --> B B -- Yes --> C B -- No --> D D --> E E --> F F --> G

Python Code

user_input = int(input("Enter a number: "))

if user_input < 0:
    print("Please enter a non-negative integer.")
else:
    factorial_result = 1
    for i in range(1, user_input + 1):
        factorial_result *= i
    print(f"The factorial of {user_input} is {factorial_result}.")
Project 7: Right Triangle Checker (right_triangle_checker.py)

Prose Description

This program determines if a triangle is right-angled by sorting three side lengths and applying the Pythagorean theorem. It demonstrates list sorting and arithmetic comparisons.

Logic Flow Diagram

flowchart TD A[Enter three side lengths] B[Sort sides in ascending order] C[Check if side1² + side2² equals side3²] D[Output: Right triangle if true; otherwise, not] A --> B B --> C C --> D

Python Code

side1 = float(input("Enter the length of side 1: "))
side2 = float(input("Enter the length of side 2: "))
side3 = float(input("Enter the length of side 3: "))
sides = [side1, side2, side3]
sides.sort()

if sides[0]**2 + sides[1]**2 == sides[2]**2:
    print("The triangle is a Right Triangle!")
else:
    print("The triangle is not a Right Triangle.")
Project 8: Number Guessing Game (my_number_guessing_game.py)

Prose Description

In this interactive game, a random secret number is generated between 1 and 100. A while loop with conditional checks provides hints (too high/too low) until the correct number is guessed.

Logic Flow Diagram

flowchart TD A[Generate random secret number (1-100)] B[Initialize guess = 0] C{Is guess ≠ secret?} D[Prompt user for guess] E{Is guess > secret?} F[Output "Too high"] G{Is guess < secret?} H[Output "Too low"] I[Exit loop when guess equals secret] J[Output congratulatory message] A --> B B --> C C -- Yes --> D D --> E E -- Yes --> F E -- No --> G G -- Yes --> H G -- No --> I F --> C H --> C C -- No --> J

Python Code

import random

secret_number = random.randint(1, 100)
guess = 0

while guess != secret_number:
    guess = int(input("Guess the number (between 1 and 100): "))
    if guess > secret_number:
        print("Too high! Try again.")
    elif guess < secret_number:
        print("Too low! Try again.")

print("Congratulations! You guessed the correct number.")
Project 9: Palindrome Checker (my_palindrome_checker.py)

Prose Description

This project checks if a word is a palindrome by reversing the word through iterative concatenation and then comparing it with the original. It highlights string manipulation and loop control.

Logic Flow Diagram

flowchart TD A[Enter a word] B[Initialize reversed_word = ""] C[While length(reversed_word) < length(word)] D[Add next character from word to beginning of reversed_word] E[Compare word and reversed_word] F[Output: Palindrome if equal; otherwise, not] A --> B B --> C C --> D D --> E E --> F

Python Code

word = input("Enter a word: ")
reversed_word = ""

while len(reversed_word) < len(word):
    reversed_word = word[len(reversed_word)] + reversed_word

if word == reversed_word:
    print(f"{word} is a Palindrome!")
else:
    print(f"{word} is not a Palindrome.")
