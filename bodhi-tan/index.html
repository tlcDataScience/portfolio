Project 1: Time Breakdown (time_breakdown.py)

Thought Process Flowchart

  • Input the total number of minutes from the user.
  • Calculate days by floor dividing by (24 * 60).
  • Calculate remaining hours by taking the remainder then floor dividing by 60.
  • Calculate remaining minutes using modulo 60.
  • Display the result.

Python Code

total_minutes = int(input("Enter the total number of minutes: "))
days = total_minutes // (24 * 60)
remaining_hours = (total_minutes % (24 * 60)) // 60
remaining_minutes = total_minutes % 60
print(f"{total_minutes} minutes is equal to {days} days, {remaining_hours} hours, and {remaining_minutes} minutes.")

Project 4: Coffee Shop Inventory

Thought Process Flowchart

  • Initialize the initial number of coffee beans (for day 1) and the daily increase.
  • Loop through days 1 to 7.
  • For each day, update the number of beans and add to the total stock.
  • Display the beans on day 7 and the total after 7 days.

Python Code

initial_coffee_beans = 50  # Day 1 beans
daily_increase = 4          # Daily increase

coffee_beans_on_day_7 = initial_coffee_beans
total_coffee_beans_after_7_days = 0

for day in range(1, 8):
    coffee_beans_on_day_7 += daily_increase
    total_coffee_beans_after_7_days += initial_coffee_beans + (day - 1) * daily_increase

print(f"Coffee beans on day 7: {coffee_beans_on_day_7}")
print(f"Total coffee beans after 7 days: {total_coffee_beans_after_7_days}")

Project 6: Factorial Calculator (factorial_calculator.py)

Thought Process Flowchart

  • Input an integer from the user.
  • Check if the number is non-negative.
  • If non-negative, multiply numbers from 1 up to the input using a loop.
  • Display the factorial result.

Python Code

user_input = int(input("Enter a number: "))

if user_input < 0:
    print("Please enter a non-negative integer.")
else:
    factorial_result = 1
    for i in range(1, user_input + 1):
        factorial_result *= i
    print(f"The factorial of {user_input} is {factorial_result}.")
Project 7: Right Triangle Checker (right_triangle_checker.py)

Thought Process Flowchart

  • Input three side lengths of a triangle.
  • Sort the sides in ascending order.
  • Check if the sum of the squares of the two shorter sides equals the square of the longest side.
  • Display whether the triangle is a right triangle.

Python Code

side1 = float(input("Enter the length of side 1: "))
side2 = float(input("Enter the length of side 2: "))
side3 = float(input("Enter the length of side 3: "))
sides = [side1, side2, side3]
sides.sort()

if sides[0]**2 + sides[1]**2 == sides[2]**2:
    print("The triangle is a Right Triangle!")
else:
    print("The triangle is not a Right Triangle.")
Project 8: Number Guessing Game (my_number_guessing_game.py)

Thought Process Flowchart

  • Import the random module and generate a secret number between 1 and 100.
  • Use a while loop to prompt the user for guesses.
  • Provide hints if the guess is too high or too low.
  • Congratulate the user when the correct number is guessed.

Python Code

import random

secret_number = random.randint(1, 100)
guess = 0

while guess != secret_number:
    guess = int(input("Guess the number (between 1 and 100): "))
    if guess > secret_number:
        print("Too high! Try again.")
    elif guess < secret_number:
        print("Too low! Try again.")

print("Congratulations! You guessed the correct number.")
Project 9: Palindrome Checker (my_palindrome_checker.py)

Thought Process Flowchart

  • Input a word from the user.
  • Initialize an empty string for the reversed word.
  • Use a while loop to build the reversed word.
  • Compare the reversed word to the original.
  • Display whether the word is a palindrome.

Python Code

word = input("Enter a word: ")
reversed_word = ""

while len(reversed_word) < len(word):
    reversed_word = word[len(reversed_word)] + reversed_word

if word == reversed_word:
    print(f"{word} is a Palindrome!")
else:
    print(f"{word} is not a Palindrome.")
