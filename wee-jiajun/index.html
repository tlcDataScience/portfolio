Wee Jia Jun's Coding Projects

Hi, I am Wee Jia Jun and I am 12 years old. I began learning robotics with LEGO Mindstorms and block‑based coding in Primary 2. I enjoy building robots to automate daily tasks and solving real‑world problems. My journey sparked an interest in Python and game development using Pygame. I have documented my adventure in robotics, coding, innovation, and STEM.

Project 1: Touch Activated Catapult

Touch Activated Catapult

I learnt how to use LEGO Technic parts to create various designs. Using conditionals (if‑else) and sensors, I programmed a catapult to launch bullets when a touch sensor is activated.

Project 2: Ultrasonic Activated Bumper Car

Ultrasonic Activated Bumper Car

I used an ultrasonic sensor to create a tractor-like robot that stops immediately when an obstacle is detected. I determined a threshold value to decide what’s too close.

Project 3: Distance Activated Shooter

Distance Activated Shooter

To play with my siblings, I built a shooter that fires bullets only when two ultrasonic sensors both detect a target. My siblings had to experiment with different distances to maximize their score.

Project 4: Arcade Gripper

Arcade Gripper

Inspired by arcade games, I built a gripper that opens when touch sensor 1 is activated and closes when touch sensor 2 is triggered. This mimics the game of picking up and releasing objects.

Project 5: Colour Activated Propellers

Colour Activated Propellers

I recreated an airplane’s propellers using LEGO Technic parts. When a colour sensor is activated, the propellers spin; a second sensor controls the spin direction using if‑else logic.

Project 6: Automated Speed Increasing Lazy Susan

Automated Speed Increasing Lazy Susan

I built an automated lazy susan using LEGO Technic parts. Each time a colour sensor detects a colour, the system increases the rotation speed—demonstrating how variables can control real‑world motions.
