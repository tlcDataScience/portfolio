Leap Year Checker (leap_year.py)

Description:

In this project, I built a program that checks if a given year is a leap year using if-else statements. The program determines whether the year is divisible by 4, and it handles special cases like century years correctly. This taught me how to control decision-making in code.

Code:

year = int(input("Enter a year: "))
if year % 4 == 0 and (year % 100 != 0 or year % 400 == 0):
    print(f"{year} is a leap year.")
else:
    print(f"{year} is not a leap year.")

Calculator Factory Production Tracker (factory_tracker.py)

Description:

I helped track the production rate of a calculator factory using a for loop. The production increases daily following an arithmetic progression. I calculated the number of calculators produced on day 30 and the total production after 30 days. This project taught me how loops can solve real-world problems!

Code:

initial_calculators = 50  # Day 1 production
common_difference = 3     # Daily increase
day_30_calculators = initial_calculators
total_calculators = 0
for day in range(1, 31):
    day_30_calculators += common_difference
    total_calculators += initial_calculators + (day - 1) * common_difference
print(f"Day 30: {day_30_calculators}")
print(f"Total after 30 days: {total_calculators}")
